## What is CelestaCX?

CelestaCX unifies every customer communication channel — voice, WhatsApp, email, social media, and web chat — into a single workspace for contact center agents, supervisors, and administrators.

## Contributing

This documentation is built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/). To run locally:

```bash
pip install mkdocs-material pymdown-extensions
mkdocs serve
```

Then open [http://localhost:8000](http://localhost:8000).
